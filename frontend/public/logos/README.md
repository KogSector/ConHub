# Company Logos

Place company logo files here with the following naming convention:

- `github.png` - GitHub logo
- `bitbucket.png` - BitBucket logo  
- `google-drive.png` - Google Drive logo
- `dropbox.png` - Dropbox logo
- `confluence.png` - Confluence logo
- `notion.png` - Notion logo
- `web-crawler.png` - Web Crawler logo (if available)

## Logo Requirements

- Format: PNG with transparent background preferred
- Size: 20x20px to 64x64px (will be scaled to 20x20px)
- Quality: High resolution for crisp display

## Fallback Behavior

If a logo file is not found, the component will automatically fall back to the original Lucide React icon.
